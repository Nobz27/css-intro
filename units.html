CSS Units and Measurements

CSS provides various units to define dimensions, spacing, and typography. These units are categorized into absolute and relative types

1. Absolute Units

Absolute units define fixed sizes that do not change relative to the viewport or parent elements

  • px- Pixels
  • cm - centimeters
  • mm - Millimeters
  • in - inches

2. Relative units --- Responsive Design

Relative units depend on parent elements, viewport size, or root elements, making them ideal for responsive design

em and rem are font related units. em unit is relative to the parent element font size and rem is relative to root element(html) font size.

Percentage(%) units are relative to the parent element (width or height)

vw(viewport width) and vh(viewport height) units are relative to the viewport available screen size
