colors and backgrounds

CSS supports multiple ways to define colors

  • Named colors eg green,blue.....
  • Hex code (Hexadecimal) #ff00ab
  • RGB -(red green blue) -- rgb(233,23,15) --allowed vales (0-255)
  • RGBA - a(alpha value - transparency level/opacity)- rgab(255,255,255,0.7)
  • HSL(HUE, Saturation , lightness) hsl(240,100%,50%)-blues
