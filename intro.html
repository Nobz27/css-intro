CSS -Cascading style sheet

Css is a programming language used to improve the presentation of a html page.

Css will help modify aspects like text sizes, colors, backgrounds, image size and shapes, borders, positions, responsiveness(different looks on different screen sizes) among other aspects of presentation.

Css is used to design user interfaces for web pages and it helps enhance user experience.

CSS syntax(how do we write css)

A selector is used to target a group of elements for styling

{} - A declaration block - groups properties and values to be applied on the selector

property - a presentation aspect you'd want to modify e.g. color, width,size, position.

Value - the final look you are interested in fot the page(red, full page, 400px......)

    
        selector{
            property: Value;
        }
    
    
Example

    
        h1{
            color: green;
        }

where do we put our css

  • inline CSS

    we use the style attribute on individual elements.( Code is less reusable and makes maintaining the application difficult)

  • Internal CSS

    Using the style element.( Doing styles within same file as structure is not recommended Styles also wont be shared to other html pages.)

  • External CSS (Recommended)

    Using a different file(.css) Linked to the html page using the link element usually placed in the head of a html page.
